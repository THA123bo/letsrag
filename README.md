## 🔍 How to Use letsrag

letsrag is designed to make local search easy and powerful.

### 1. Add Your Data

- You can add PDF files, text documents, or other data to search.  
- Use the interface to select folders or files to include.  
- letsrag processes your files to create indexes.  

### 2. Run Hybrid Search

- letsrag uses hybrid search. This means it applies keyword matching and semantic search together.  
- It finds relevant answers faster and more accurately.  

### 3. Use the Reranker

- The reranker improves search results by sorting the best answers on top.  
- You do not need to do anything. letsrag handles reranking automatically.  

### 4. Generate Answers with HyDE

- HyDE (Hypothetical Document Embeddings) helps letsrag generate better responses by understanding the context of your query.  
- This happens behind the scenes for you.

### 5. Review Your Results

- letsrag shows answers in a clear list.  
- You can click on results for more details.  
- If you want, you can adjust search settings from the menu.
