## 2. Architecture Principles
The core philosophy is integrating deep learning components (the localized 7B parameter model) with explicitly defined, deterministic knowledge graphs (Symbolic logic). This forces the generative model to obey absolute mathematical and structural realities rather than merely guessing the most probable next word based on flawed, filtered context.

### 2.1 Advanced Data Flow Pipeline
1. **Query Ingestion & Symbolic Mapping:** The user query is expanded using standard LLM logic, but simultaneously mapped against a 5-trillion edge ontological Knowledge Graph to identify hard relationships (e.g., *Is 'Class A' a subnet of 'System B'?*).
2. **Deterministic Retrieval:** A hybrid workflow triggers: Semantic Search (ChromaDB) and BM25 (Lexical) fetch unstructured text. Concurrently, a GraphQL traversal identifies unquestionable logical constraints.
3. **Graph-Constrained Synthesis:** The 7B model is fed a highly structured prompt containing both the raw text chunks and a JSON-formatted list of mandatory logical constraints. The model cannot output an answer that contradicts the graph.
